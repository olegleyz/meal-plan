# База ужина

## Принцип

Стабильный белковый якорь + овощной объём + углеводный модуль по контексту.  
Большинство ужинов — из заготовок недели, сборка ≤ 15 мин.

---

## Архитектура

```
Dinner = Protein Base + Vegetable Volume + Context Carb
```

---

## Protein Base — обязательно

Выбрать один:

| Вариант | Порция | ккал | Белок |
|---|---|---|---|
| Куриная грудка / бедро без кости | 150–180г | 230–280 | 35–42г |
| Лосось (запечённый / Costco) | 150г | 270 | 30г |
| Треска / морской окунь | 200г | 160 | 40г |
| Консервированный тунец | 130г | 150 | 30г |
| Тунец + варёное яйцо | 130г + 1 шт | 220 | 36г |

---

## Vegetable Volume — обязательно, выбрать 1–2

| Вариант | Порция | ккал |
|---|---|---|
| Запечённые овощи (перец, баклажан, кабачок) | 200–300г | 60–100 |
| Брокколи / пак-чой / шпинат (паровые) | 200г | 50 |
| Зелёный салат (руккола, шпинат) + помидор | 100г | 25 |
| Суп-основа (тыква / чечевица) | 300мл | 150–200 |

---

## Context Carb — по типу дня

| Тип дня | Углеводный модуль | Порция | ккал |
|---|---|---|---|
| День отдыха | нет / белая фасоль как добавка | — | — |
| Лёгкий бег (пн, пт) | батат / картофель | 150г | 120–130 |
| Утренний зал (вт, чт) | гречка / пшено | 60г сухих | 200 |
| Pre-run (ср, 17:30) | паста / рис | 80г сухих | 280 |
| HIIT + yoga / бокс (сб, вс) | корнеплоды / крупа | 60г | 180–200 |

---

## Итого по типу дня

| Тип дня | ккал | Белок |
|---|---|---|
| Отдых / лёгкий бег | ~430–520 | ~45–52г |
| Утренний зал | ~530–600 | ~42–50г |
| Pre-run (ср) | ~580–650 | ~43–48г |
| HIIT / бокс (сб, вс) | ~550–630 | ~48–55г |

---

## Адаптация под каждый день недели

| День | Protein Base | Context Carb | Особенность |
|---|---|---|---|
| Пн | Курица (из заготовки) | Батат / нет | До йоги 20:30 — не перегружать |
| Вт | Курица (из заготовки вт) | Гречка | Gym day, белок важен |
| Ср | Лосось (из заготовки ср) | Паста | Pre-run, углеводы нужны |
| Чт | Курица (из заготовки вт) | Пшено | Gym day, белок важен |
| Пт | Треска / окунь | Картофель | Рыбный день, лёгкий |
| Сб | Курица (из субботнего жаркого) | Корнеплоды | После HIIT+йоги |
| Вс | Тунец + яйцо + фасоль | Нет / минимум | Лёгкий финал недели |

---

## Инварианты

* Белок ≥ 40г — обязательно каждый день.
* В среду — углеводный компонент обязателен (паста/рис).
* Перед сном — нет жирных соусов, жареного, острого.
* Порция крупы не превышает 80г сухого веса.

---

## Продукты в ротации (что держать готовым)

* Курица запечённая (вт/ср заготовка → ужин вт, чт)
* Лосось Costco запечённый (ср заготовка → ужин ср)
* Запечённая курица целиком (сб → ужин сб + об вс)
* Треска / морской окунь замороженные
* Консервированный тунец
* Гречка и пшено варёные (заготовка вт)
* Запечённые овощи (воскресная заготовка)
