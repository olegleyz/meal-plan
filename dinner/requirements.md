# Требования к ужинам

## 1. Цель компонента

Ужин должен:

* завершать белковый баланс дня (поддержка MPS за ночь),
* не нарушать качество сна,
* вписываться в калорийный бюджет после завтрака и обеда,
* адаптироваться под тип дня и вечернюю активность,
* быть операционно простым — преимущественно из заготовок.

---

## 2. Нефункциональные требования

### Надёжность

* В будни — из заготовок или простой сборки (≤ 15 мин).
* Не требовать решений о составе в момент усталости.

### Повторяемость

* 70–80% ужинов — из ограниченного набора паттернов (курица / рыба / бобовые + овощи).
* Вкус и профиль могут меняться, макро-паттерн — предсказуем.

### Наблюдаемость

После ужина должно быть легко оценить:

* качество сна (засыпание, глубина),
* утреннее восстановление,
* ощущение голода/тяжести перед сном.

---

## 3. Функциональные требования

### FR-1. Белковый минимум

Каждый ужин должен содержать **≥ 40 г белка**.  
Оптимальный диапазон: **40–55 г**.

Обоснование: казеин-содержащие белки (рыба, курица, творог) поддерживают MPS в течение ночи. Ужин — последний белковый приём, от него зависит overnight recovery.

### FR-2. Контроль калорий

| Тип дня | Диапазон ужина |
|---|---|
| День отдыха / лёгкая пробежка | 400–550 ккал |
| Тренировочный день (утренний зал / HIIT) | 500–650 ккал |
| Pre-run день (ужин ДО пробежки) | 550–700 ккал (нужны углеводы) |

### FR-3. Совместимость со сном

Ужин **не должен**:

* вызывать тяжесть → нет жирных соусов / жареного в масле,
* провоцировать рефлюкс → нет острого + лёжа сразу,
* создавать избыток углеводов → нет сладкого финала.

### FR-4. Углеводы по контексту

| Тип дня | Углеводный модуль |
|---|---|
| Отдых / лёгкий бег | минимальные (овощи, зелень) |
| Утренний зал | умеренные (гречка, пшено, батат) |
| Pre-run (ср, 17:30) | повышенные (паста, рис) — нужна энергия для бега |

### FR-5. Пищеварительная переносимость

* Ужин за **4–5 часов до сна** — время усвоения достаточно.
* Исключение: среда — ужин в 17:30, через час trail run → лёгкое, хорошо усваиваемое.
* Не перегружать ЖКТ перед ночью: без избытка клетчатки / бобовых на ночь.

---

## 4. Архитектурный паттерн

```
Dinner = Protein Base + Vegetable Volume + Context Carb (optional)
```

### Protein Base — обязательно

Должен давать **≥ 35–40 г белка сам по себе**:

* Курица (грудка / бедро без кости)
* Рыба (лосось, треска, окунь, тунец)
* Комбо: яйца + бобовые

### Vegetable Volume — обязательно

* Запечённые овощи / зелень / суп-основа
* Высокий объём, низкая калорийность, клетчатка, микронутриенты

### Context Carb — по типу дня

* Гречка / пшено / батат / паста / картофель
* Или не подключается (отдых)

---

## 5. Инварианты

* Белок ≥ 40 г в каждом ужине.
* Перед trail run (ср) — углеводный компонент обязателен.
* После тяжёлого зала (вт, чт) — белок важнее углеводов.
* Ужин не должен быть слишком лёгким: < 400 ккал → риск ночного голода.
* Ужин не должен быть тяжёлым: > 700 ккал → риск нарушения сна.

---

## 6. Антипаттерны

| Код | Название | Проблема |
|---|---|---|
| AP-1 | Слишком лёгкий ужин | < 400 ккал, < 35г белка → голод ночью, плохой overnight MPS |
| AP-2 | Углеводный финал | Паста / рис без достаточного белка → инсулиновый всплеск перед сном |
| AP-3 | Жирная бомба | Жареное + соус → тяжесть, рефлюкс, плохой сон |
| AP-4 | Пропуск белка в gym day | После зала белок нужен и на завтраке, и на ужине |

---

## 7. Приёмочные критерии

Хороший ужин:

- [ ] содержит ≥ 40 г белка
- [ ] вписывается в калорийный бюджет дня
- [ ] не вызывает тяжести и нарушения сна
- [ ] содержит овощной компонент
- [ ] углеводы соответствуют типу дня
- [ ] готовится ≤ 15 мин (или из заготовки)

---

## 8. Приоритеты

| Приоритет | Требование |
|---|---|
| **P0** | ≥ 40г белка, вписывается в бюджет, совместим со сном |
| **P1** | овощи, правильные углеводы по типу дня |
| **P2** | вкусовая вариативность, сезонность |

---

## Сжатая версия

> Ужин должен содержать ≥ 40г белка, завершать дневной калорийный бюджет без превышения, не нарушать сон, адаптировать углеводный компонент под тип дня; ключевой инвариант — белок для overnight recovery обязателен, особенно в тренировочные дни.
